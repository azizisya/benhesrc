Terrier Future Features and Known Issues

List of features and known issues that are marked for future Terrier versions:

Future Features

  • Move to single-pass indexing by default.
  • Ant task for compiling Terrier.
  • Terrier's own Exceptions for setup, indexing, querying.
  • Postscript parsing (via PDF?).
  • Refinement of desktop search application with improved interface and parsers for more types of documents, as well as better integration with common Operating Systems.
  • Move towards more systematic updatable indices.

Known Issues

  • Open With dialog for Desktop Terrier on Mac OS X.
  • Faster PDF parsing, perhaps calling pdf2text if available.
  • Refinement of the query language: remove the ambiguity warnings when generating the parser with ANTLR, and introductions of a more structures query language.
